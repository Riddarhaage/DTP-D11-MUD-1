\doxysubsection{Detailed Description}
A class for opening doors between \mbox{\hyperlink{classdtp11___m_u_d__1_1_1_room}{Room}}\+:s, the Lockkey is defined for two room numbers, and it can be adressed whether it really opens the door from one \mbox{\hyperlink{classdtp11___m_u_d__1_1_1_room}{Room}} to another. 

{\bfseries{weakness\+:}} the key cannot be used to actually open a door from \mbox{\hyperlink{classdtp11___m_u_d__1_1_1_room}{Room}}, the \mbox{\hyperlink{classdtp11___m_u_d__1_1_1_room}{Room}} code must detect whether the Lockkey fits a certain door, and then open it itself.
