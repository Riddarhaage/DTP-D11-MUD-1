Detailed Description

A class for opening doors between Room:s, the Lockkey is defined for two room numbers, and it can be adressed whether it really opens the door from one Room to another.

weakness: the key cannot be used to actually open a door from Room, the Room code must detect whether the Lockkey fits a certain door, and then open it itself.
